# Computing History AI Agent
Built on Microsoft Azure AI Foundry

## Overview
A domain-specific AI agent that answers questions about computing history,
powered by GPT-4.1-mini and web search via Azure AI Foundry.

## Tech Stack
- Azure AI Foundry (Agent Builder)
- Python, Flask, OpenAI Responses API
- Azure AI Projects SDK, Azure Identity (Entra ID)
- VS Code Foundry Toolkit, GitHub Copilot

## Features
- Multi-turn conversation with history tracking
- Web search tool for real-time information
- Flask web UI + Python CLI client
